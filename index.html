Was ist HTML?

Die Grundlage jeder Webseite

Wenn du im Browser eine Webseite öffnest, lädt im Hintergrund eine Textdatei herunter. Diese ist in HTML geschrieben – der Sprache, die dem Browser erklärt, wie der Inhalt aufgebaut ist.

HTML steht für HyperText Markup Language. Stell dir vor, du nimmst einen Text und markierst mit Stiften: was ist eine Überschrift, was ein Absatz, was eine Liste. Genau das macht HTML – nur für den Browser.

SpracheZuständig für …Beispiel
HTMLStruktur & Inhalt„Das ist eine Überschrift"
CSSAussehen & Design„Die Überschrift ist blau, 24 px groß"
JavaScriptInteraktivität„Bei Klick passiert etwas"
ℹ️ In diesem Kurs
Wir lernen HTML und machen einen ersten Einstieg in CSS. JavaScript ist ein eigenes, weiterführendes Thema.

Werkzeuge – was brauche ich?

HTML-Dateien sind ganz normale Textdateien. Du brauchst nur einen Texteditor zum Schreiben und deinen Browser zum Anschauen.

✅ Empfehlung
Editor: Visual Studio Code (kostenlos) – hebt deinen Code farbig hervor.
Browser: Firefox, Chrome, Edge oder Safari.

Speichere Dateien immer mit der Endung .html und stelle die Zeichenkodierung auf UTF-8 – damit werden Umlaute richtig angezeigt.

Überschriften

Es gibt sechs Überschriften-Ebenen: <h1> (die größte) bis <h6>. Das „h" steht für englisch heading. Jede Seite sollte genau eine <h1> haben.

<h1>Das ist die Hauptüberschrift</h1> Anfangs-Tag            Inhalt                  End-Tag (mit /)
überschriften.html
<h1>Die Hauptüberschrift</h1>
<h2>Ein Abschnitt</h2>
<h3>Ein Unterabschnitt</h3>
⚠️ Ebenen nicht überspringen!
Nach <h2> kommt <h3> – nicht direkt <h4>. Suchmaschinen und Screenreader brauchen die korrekte Reihenfolge.

Text in Absätzen

Ein Zeilenumbruch in deiner Datei bedeutet dem Browser nichts – er fasst alles in eine Zeile. Du musst ihm mit <p> (von englisch paragraph) zeigen, wo ein Absatz beginnt und endet.

absaetze.html
<h1>Schreinerei Meier</h1>
<p>Willkommen auf unserer Webseite!</p>
<p>Wir sind seit Jahren auf Holzmöbel spezialisiert.</p>
ℹ️ Semantik
Durch korrekte Tags machst du die Bedeutung deines Inhalts für Browser, Suchmaschinen und Screenreader verständlich. Das nennt man semantisches HTML.

Das HTML-Grundgerüst

Eine vollständige HTML-Datei braucht einen festen Rahmen. Der <head>-Bereich ist unsichtbar und enthält Informationen für den Browser.

grundgeruest.html
<!doctype html>
<html lang="de">
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Schreinerei Meier</title>
  </head>
  <body>
    <!-- Sichtbarer Inhalt hier -->
    <h1>Willkommen!</h1>
  </body>
</html>

<meta charset="utf-8">Umlaute werden korrekt angezeigt.
<title>Text im Browser-Tab.

Navigation

Für das Hauptmenü einer Webseite gibt es das <nav>-Element mit einer Liste von Links.

navigation.html
<nav>
  <ul>
    <li><a aria-current="page">Startseite</a></li>
    <li><a href="leistungen.html">Leistungen</a></li>
    <li><a href="kontakt.html">Kontakt</a></li>
  </ul>
</nav>
ℹ️ aria-current="page"
Markiert die aktive Seite im Menü – wichtig für Screenreader. Bei diesem Eintrag lässt man href weg (man verlinkt nicht auf die Seite, auf der man schon ist).

Gestaltung mit CSS

Farben · Schriften · Abstände

Was ist CSS?

HTML baut das Haus, CSS streicht die Wände. CSS steht für Cascading Style Sheets. Ohne CSS erscheinen Inhalte sehr schlicht. Mit CSS kontrollierst du Farben, Schriftarten, Abstände und Layouts.

ℹ️ Browser-Standardstile
Jeder Browser hat eingebaute Grundstile – deshalb wirken Überschriften auch ohne CSS fett und groß. Mit CSS überschreibst du diese und legst eigene Regeln fest.

CSS-Regeln schreiben

Eine Regel besteht aus Selektor (welches Element?), Eigenschaft (was?) und Wert (wie?).

h2 {
  color: darkblue;
  font-size: 1.5em;
} Selektor       Eigenschaft: Wert;     → gilt für alle <h2>

font-weightboldFettschrift

Elemente nebeneinander mit CSS Grid

Mit display: grid verwandelst du einen Container in ein Raster und kannst Spalten frei definieren.

style.css
.layout {
  display: grid;
  grid-template-columns: 1fr 200px;
  /* 1fr = restlicher Platz · 200px = feste Breite */
  gap: 20px;
}
ℹ️ id vs. class
id: einmalig – im CSS mit #name ansprechen.
class: mehrfach verwendbar – im CSS mit .name ansprechen.
